## Features

- Fully **responsive admin dashboard** with **4 cards per row** on large screens.
- **Role Management**: Create, edit, and manage user roles.
- **Products Management**: Add, list, and manage products with images.
- **Orders Management**: View, search, and create orders with validations.
- **Support Tickets & Messages**: Track tickets and messages efficiently.
- **Settings & Integrations**: Manage system settings and external integrations.
- **Calendar & System Logs**: View events and system activity logs.
- **Professional UI**: Uses MUI for a modern, consistent interface with hover effects.
- **Icons**: Uses **Tabler Icons** for clean and scalable icons.
- **Routing**: Client-side routing using **React Router DOM**.

---

## Technologies Used

- **React.js** – Frontend library for building interactive UI  
- **Material-UI (MUI)** – UI components and design system  
- **React Router DOM** – Client-side routing  
- **Tabler Icons** – Professional SVG icons  
- **Dummy Data** – For products, roles, and orders demonstration
